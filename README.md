# finalizacao-libras-ai
Etapa de finalização do projeto Libras AI, responsável pela geração do vídeo com janela de Libras.

## 🛠️ Tecnologias (Sprint 1)
- HTML (protótipo)
- CSS (layout básico)

## 📌 Observações
Este projeto está em fase inicial e utiliza uma abordagem simplificada para demonstrar a etapa de finalização.

## 🎥 Funcionalidade esperada
Exibir um vídeo com uma janela de Libras sobreposta (overlay), simulando a acessibilidade para usuários com deficiência auditiva.
